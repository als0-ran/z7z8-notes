# less

---

1. moving

   * **e, y**： next/back line
   * **f, b, z, w**:  next/back page
   * **数字** ： next N line
2. searching

   * **/pattern**: search forward
   * **?pattern**: search backward
     ! no matching
3. jumping

   * **g/G** first/last
   * **p,%** N precent of file
   * **m,N** top/bottom mark letter
   * **'** goto mark letter
